1. Навигация по файловой системе. Попрактиковаться в перемещении между каталогами, используя полный и относительный путь. 
Перечислить, какие параметры команды cd позволят быстро вернуться в домашний каталог, позволят перейти на уровень выше. 

***Навигация по файловой системе***
ls / -содержимое папок (/ - корень)
ls –l / - подробное описание содержимых папок (с типами данных)
ls –l –a / (ls –la /) - скрытые файлы (-a). Все файлы начинающиеся на точку, считаются скрытыми
ls –la ./ - на текущую директорию
ls -l dir* - покажет содержимое директорий, начинающихся на dir

Абсолютный адрес ls -l /var/log. Если мы находимся в корне.
Относительный адрес ls -l ../../var/log. Перешли на уровень выше (home), после еще на уровень (корень)

***cd - change directory***
cd .. - переход на уровень выше
cd (cd ~)  - возврат на домашнюю директорию
cd -   - возврат в предыдущую директорию(в которой были до этого)


2. Управление файлами и каталогами и текстовые редакторы. Создать файл с наполнением, используя несколько способов. 
Использовать разобранные текстовые редакторы для наполнения файлов данными. Создать копии созданных файлов, создать несколько каталогов с подкаталогами, 
перенести несколько файлов в созданные каталоги. Перечислить команды и используемые параметры команд.

touch test_file – создание файла
touch file {1..99}, touch file {a..z}, touch file {1..99}{a..z} – способы создания множества файлов
rm test_file – удаление файла

mkdir test_dir – создание директории
mkdir -p test_dir/dir1 / mkdir -p test_dir{1..5}/dir{1..2}- создание каталога с подкаталогами
rmdir (rm -r) – удаление директории
rm –rf  ./* - удаление всех файлов в текущей директории
rm file??? – удалить файлы, у которых 3 символа на конце
rm file?7? – удалит файлы с данным условием (в конце символ, предпоследний символ - "7")

cp file1 file2 – копирует file1 и создает file2
cp –r dir1 dir2 – копирует директорию dir1 и создает dir2

mv file1 file3 – переименовывает file1 в file3
mv dir1 dir2 - переименовывает dir1 в dir2
mv file1 dir2/file2 – перемещает file1 в директорию dir2
mv dir1/*dir2 - перемещение всех файлов из dir1 в dir2

vim my_file – создание файла и наполнение с помощью vim
nano my_file2 – создание файла и наполнение с помощью nano

cat /etc/passwd  (less /etc/passwd) - чтение файла
cat my* - чтение всех файлов, начинающихся на «my»
